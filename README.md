# TokoBus
## 概要
早稲田大学所沢キャンパスの非公式スクールバス時刻表アプリです。  

## 主な機能
* 曜日や祝日などその日に合ったダイヤを自動で表示します。
* 時刻表と共に残り時間も表示されるため、どのバスに乗れるか簡単に確認できます。
* フロンティアリサーチセンターへのシャトルバスにも対応しています。
