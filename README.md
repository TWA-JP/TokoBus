# TokoBus
早稲田大学所沢キャンパスのスクールバス時刻表アプリです。   
曜日や祝日などその日に合ったダイヤを自動で表示します。  
また時刻表と共に残り時間も表示されるため、どのバスに乗れるか簡単に確認できます。
フロンティアリサーチセンターへのシャトルバスにも対応しています。
