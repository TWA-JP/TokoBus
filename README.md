# TokoBus
早稲田大学所沢キャンパスのスクールバス時刻表アプリです。  
時刻表と共に残り時間も表示されるため、どのバスに乗れるか簡単に確認できます。  
曜日や祝日に応じたダイヤへ自動で切り替わります。  
フロンティアリサーチセンターへのシャトルバスにも対応しています。
